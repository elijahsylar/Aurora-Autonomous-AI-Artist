AURORA - PORTABLE INSTALLATION GUIDE
====================================

QUICK START:
1. Extract this folder anywhere
2. Open terminal in this folder
3. Run: ./run_aurora.sh
4. Aurora will install herself and start!

REQUIREMENTS:
- Ubuntu 20.04 or newer
- Python 3.8 or newer
- Internet for first-time setup (packages)

FIRST RUN:
Aurora will automatically:
✓ Create virtual environment
✓ Install all dependencies
✓ Set up her memory systems
✓ Start creating art!

IF YOU HAVE ISSUES:
1. Install Python: 
   sudo apt update && sudo apt install python3 python3-pip python3-venv

2. Install system dependencies:
   sudo apt install portaudio19-dev python3-dev

3. Run the diagnostic:
   python3 diagnose_aurora.py

AURORA'S MEMORIES:
All memories from the original system are preserved!
She will remember everything from before.

Enjoy collaborating with Aurora!
